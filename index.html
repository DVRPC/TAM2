    Transportation Assets Management Tool for Municipal Roads

    •  About this tool
    •  User's Guide
    •  Disclaimer
    •  Contact

    The Transportation Assets Management (TAM) Tool for the Tri-County Region is interactive, web-based, and secure. The Tool can be used on desktop and laptop computers, and portable devices using Google Chrome, Mozilla Firefox, or Apple's Safari as a web browser. [Internet Explorer is not fully compatible with the Tool, and not recommended.] Passwords to access the Tool have been supplied to the client municipalities.

    The Tool is structured in two parts. This is Part 2.

    • Part 1 is the data editor. Part 1 uses ESRI's ArcGIS-online as its platform for point-and-click access to enter, edit, or query local road and bridge information contained in a centralized project database. The project database supporting Part 1 is pre-loaded with some physical attribute data - which is current to the date of the parent database from which it was drawn, including:
      • PennDOT's Roadway Management System (RMS) database from September 2013,
      • PennDOT's Bridge Management System (BMS) database from September 2013, and
      • PennDOT's Local Roads and Bridges (LR&Bs) database from January 2013.

      Structural condition ratings for local roads were determined and input by municipal staff, and are current for the year specified by the host municipality. Local road and bridge ownership information reflects the input of the host municipality.

    • Part 2 is the analytical component. Part 2 uses Google Maps with customized functionality for analyzing (manipulating and processing) local road conditions including: selecting improvement treatment types and additional options, applying cost-estimating functions and project scheduling features. Part 2 also incorporates capabilities for transferring information into spreadsheet and word processing-software for more robust analysis and reporting, proposals, and cost estimates.

    • Parts 1 and 2 are linked via the internet for updates.
    A User's Guide, or operating manual, on the Tool has been prepared (DVRPC Publication No. PM15032) and provided to the client municipalities. A PDF of the User's Guide is accessible via the Tool. Additional support is available by contacting DVRPC. Any errors in the data should be brought to DVRPC's attention.
